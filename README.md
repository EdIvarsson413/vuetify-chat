# VuetifyChat

Este es un chat simple, solo acepta texto. Proyecto hecho en Vite + Vue

## Estilos

Hecho en VueJS con estilos de Vutify en su versión 3

## Instalar dependencias

```sh
npm install
```

### Compilacion para desarrollo

```sh
npm run dev
```

### Compilacion para producción

```sh
npm run build
```
